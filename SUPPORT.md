# Support

Thank you for using Integr8sCode! We want to ensure you have a smooth experience. This document outlines how to get help, report issues, and suggest improvements.

## Where to Get Help

### 1. Documentation
Before asking a question, please check our project documentation (located in the `docs/` directory or our documentation site). It covers architecture, deployment, and configuration details.

### 2. GitHub Discussions
For general questions, best practices, or architectural discussions that aren't bugs, please use [GitHub Discussions](https://github.com/HardMax71/Integr8sCode/discussions).

### 3. GitHub Issues
If you encounter a bug or have a specific feature request, please verify it hasn't already been reported, then [open an issue](https://github.com/HardMax71/Integr8sCode/issues).

*   **Bug Reports:** Please use the "Bug Report" template and provide reproduction steps.
*   **Feature Requests:** Please use the "Feature Request" template and describe the use case clearly.
